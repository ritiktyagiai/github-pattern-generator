About GitHub Pattern Generator

The GitHub Pattern Generator is a lightweight, interactive open-source tool designed to help developers personalize their GitHub profile contribution graphs. GitHub's contribution graph is a calendar that tracks commits, issues, and pull requests over the past year. By adjusting the timing and count of commits, developers can write words, draw patterns, or design symbols that display directly on their profile graphs.

Normally, designing contribution graph artwork requires manually calculating days, weeks, and contribution values, which can be a slow and error-prone process. The GitHub Pattern Generator replaces this complexity with a visual contribution grid canvas representing the standard 53-week, 7-day contribution grid. Developers can click or draw on cells, immediately preview the design, and export it as a pattern.json configuration.

Core Features

  • Visual Editor: Left-click to add a contribution block, and right-click to clear a block. Easy to use, highly responsive canvas.
  • Instant Preview: See your graph design come to life in real-time, supporting light and dark mode viewports.
  • JSON Export: Save your patterns locally as a structured pattern.json configuration file.
  • Automation Ready: Compatible with automated scripting tools to schedule commits on selected dates according to your design.
